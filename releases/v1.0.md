# 聽雪居 v1.0

第一個上傳到 GitHub 的可用版本。

## 這版有什麼

- Android 12+ 原生聊天 App。
- 支援 OpenRouter、Groq、Cerebras 與自訂 OpenAI 相容端點。
- 可在手機本地保存聊天紀錄。
- 支援模型選擇與模型搜尋。
- 支援角色、Persona、世界設定集。
- 可從 TXT、JSON、DOCX 文件交給 AI 整理成角色、Persona 或世界設定草稿。
- 長對話遇到上下文過長時，可自動摘要舊內容並重試。
